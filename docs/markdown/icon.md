# ICON 图标

----

语义化的矢量图形

### 如何使用

使用 `class="icon"` 来声明图标，具体图标的名称请 `copy` 相应的标签

:::demo

```html
<i class="icon icon-zoom-in"></i>
<i class="icon icon-zoom-out"></i>
<i class="icon icon-circle-correct"></i>
<i class="icon icon-circle-cancel"></i>
```

:::
